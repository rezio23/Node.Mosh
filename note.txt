Node Module Notes
=================

1. Every file in a Node.js app is a module.
   Example: `app.js`, `logger.js`

2. Variables and functions declared in one file are scoped to that file.
   They are not available in other files by default.

3. To share code between files:
   - Export from one module.
   - Import/require it in another module.

Example
-------
In `logger.js`:
`module.exports = log;`

In `app.js`:
`const log = require('./logger');`


Listing All Dependencies
=================
1. npm list or npm list --depth=0



View Registry Info for a Dependencies Package
=================
1. npm view dependencies_name or npm view dependencies_name dependencies
